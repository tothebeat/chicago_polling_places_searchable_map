About

There are 1484 polling locations for the 2015 Chicago Mayoral election taking place on February 24, 2015. The Chicago Board of Election Commissioners provides a search form on their website for finding the polling location nearest you, but they don't have a map of all of the locations. Instead, they provide a PDF with the polling locations broken down by ward and precinct in text form only.

The polling location data was extracted from the PDF using this Python script created by Nick Bennett at Hoy Chicago.

This is built using Derek Eder's fantastic Searchable Map Template using Google Fusion Tables.
